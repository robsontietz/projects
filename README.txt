Intalação do aplicativo:

1 - Copie o arquivo "Marvel.apk" para um dispositivo móvel com sistema Android

2 - Pelo dispositivo, localize e execute o arquivo

3 - Se o dispositivo questionar se o arquivo é confiável clique em "sim"

4 - Um atalho da aplicação foi criado, clique no mesmo para abrir a aplicação


Sobre o aplicativo:

1 - O App lista meus 3 heróis favoritos e exibe a informação de quantos histórias já participou

2 - Ao clicar em um deles é listado as 5 primeiras histórias aos quais cada um consta

3 - Gerei tudo em apenas 1 tela simples, conforme solicitado

4 - A linguagem utilizada foi Flutter/Dart

5 - Bônus: no menu é possível acessar uma lista e pesquisar por qualquer outro herói da Marvel
